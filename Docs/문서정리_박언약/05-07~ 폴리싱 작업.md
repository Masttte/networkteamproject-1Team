## VFX

### 스크린 이펙트
- VolFx - Jpeg 레퍼런스를 RnD 하여 스크린 이펙트를 제작 (URP에 Vol Fx 추가)
- 단순히 에셋을 적용하는 것이 아닌, 레퍼런스 분석을 통해서 원하는 효과를 구현
- CustomScreenVol(글로발 볼륨), CustomScreenPass(렌더 패스), CustomScreen(셰이더) 코드 작성
```
CustomScreenVol
    VolumeComponent, IPostProcessComponent 를 상속받아 Volume Component Menu에서 Add Override로 추가할 수 있도록 구현
CustomScreenPass
    Jpeg라는 랜더패스를 베이스로 첫 분석 작업을 하였으며
    필요한 부분만 덜어낸후 AI에게 빼곡히 주석을 작성하라고 하여 분석에 도움이 되었다
    컬러풀한 색깔조절을 추가하여 원하는 효과를 만들어냈다 
CustomScreen.shader
    셰이더는 렌더 패스에서 설정한 파라미터를 받아서 화면에 적용하는 역할을 한다
    레퍼런스 코드와 비슷하지만 따라작성하는데 의의를 둠
```
