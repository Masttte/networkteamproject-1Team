## TeamManager
팀에 따라 다른 프리팹을 스폰하고, 공통 로직과 팀 전용 로직을 분리

---
## TeamBase
- `OnValueChanged` 구독으로 팀 변경 → 카메라 컬링(오너) → `LocalManager.IamB` 동기화 → `OnTeamSetup()`(모두)
- 레이어 분리는 `ApplyCullingMask`에서 처리

## TeamA (A팀 / 시민)
**프리팹 구조:**
```
PlayerA  (Layer: Player)
├─ ModelHuman   (Layer: Average 10)    ← A 시점: 보통 사람
├─ ModelMonster (Layer: Beautiful 11)  ← B 시점: 괴물 (기본 비활성화)
└─ CamPoint
```

**B클라이언트 Avatar 교체 처리:**
- Inspector에서 `_modelHuman`, `_modelMonster` 두 자식 오브젝트를 참조
- `OnTeamSetup` 호출 시 `LocalManager.Instance.IamB` 가 true이면 `ApplyMonsterAvatar()` 실행
- 오브젝트 활성화/비활성화 처리와 루트 아바타를 몬스터 아바타로 교체

## TeamB (B팀 / 마피아)
- 모두에게 사람으로 보이는 기본 프리팹 사용
- 추후 확장 가능
